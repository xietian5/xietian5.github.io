## 记录git常用命令, 持续更新

基本命令参考: [git简易指南](http://www.bootcss.com/p/git-guide/)



# clone远程仓库

ssh:

`git clone git@github.com:USERNAME/XXX.git`

https:

`git clone https://github.com/USERNAME/XXX.git`

# 同步远程仓库

`git pull`

# 添加与提交

`git add <filename>`

`git add *`

`git commit -m "备注"`

`git push origin master`

# 删除仓库文件

以下命令会删除本地文件以及服务器文件

`git rm <filename>`

`git rm *`

`git commit -m "备注"`

`git push origin master`
